# Loader

A stateless animated component that alerts a user to wait for an activity to complete.

## Naming

The _Spinner_ as name is more widely used, however we call it _Loader_ as it is not required to "spin" in all themes. Some theme may have a different animation. For example:

- https://loading.io/css/
- http://tobiasahlin.com/spinkit/

## Properties

| Name          | Description                                                                                                |
| :------------ | ---------------------------------------------------------------------------------------------------------- |
| color         | Defines a color of a loader, [common property](/specifications-common-properties#color).                   |
| labelPosition | Defines a label position around the matching indicator, possible values: `above`, `below`, `start`, `end`. |
